# Welcome to the Custom Tools Repository

Welcome to the documentation for my collection of custom automation scripts and tools. These tools are designed to streamline various administrative and file management tasks.

## Global Dependencies
The following software is required to run **any** tool in this repository:

* **Operating System**: Microsoft Windows 10 or 11.
* **Environment**: Windows PowerShell 5.1 or later.

## Available Tools

| Name | Description | Dependencies |
| :--- | :--- | :--- |
| [[pptx to pdf]] | recursively scans a directory for `.ppt` and `.pptx` files and converts them to PDF, mirroring the source folder structure in the destination. | Microsoft PowerPoint (Desktop App) |
| [[Zip Folders Individually]] | Scans a specific directory and compresses each immediate sub-folder into its own independent `.zip` archive. | 7-Zip |
